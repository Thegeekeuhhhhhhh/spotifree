# spotifree

## Installation
```
npm install
pip install PyInstaller
pip install -r requirements.txt
```

## What is Spotifree ?

Spotifree is a desktop application that allows you to play your favorite songs, using Youtube browsing

## What can be found on Spotifree ?

Spotifree is splitted in different parts

### Home
While on the home page, you can look at all the songs you ever played

### Search
While on the search page, you can enter the name of a song, an artist or just a youtube video, podcast...
Then, you will have a list of results on which you can click to play songs or to add them to any playlists, including your Liked Songs

### Library
While on the library page, all your playlists are displayed including your Liked Songs, you can then click on any of them to access to its content.

### Playlists
A playlist is a feature that allows you to create a list of songs. :)
